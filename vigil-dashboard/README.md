# VIGiL Dashboard

> "Intelligence That Protects."

This is the frontend dashboard for Project VIGIL —
built with React + TailwindCSS.

## Team
**Navadhara** | Far Away 2026

## Run
npm install
npm run dev

## Features
- Live railway signal health chart
- Real-time alert feed with severity classification
- Agent activity log — WatchAgent, AlertAgent, ActionAgent
- Active train monitoring strip
- OrbitMind module slot — architected for Round 2
